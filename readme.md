2.小程序文件介绍：
 app.js: 注册小程序全局应用
 app.json: 注册小程序页面配置路径，通过pages指定
 创建一个文件的时候需要由四个文件组成：
    a.XX.js：小程序逻辑/数据通信
    b.XX.wxml：小程序展现
    c.XX.wxss : 小程序样式
    d.XX.json： 配置相关
